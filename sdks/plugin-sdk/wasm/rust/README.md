# StreamKit WASM Plugin SDK

SDK for writing WASM (Component Model) StreamKit plugins.

## Install

```toml
[dependencies]
streamkit-plugin-sdk-wasm = "0.1.0"
```

See the docs site for a full example: https://streamkit.dev/guides/writing-plugins/
