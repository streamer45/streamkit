# StreamKit Native Plugin SDK

SDK for writing native (C ABI) StreamKit plugins.

## Install

```toml
[dependencies]
streamkit-plugin-sdk-native = "0.1.0"
```

See the docs site for a full example: https://streamkit.dev/guides/writing-plugins/
